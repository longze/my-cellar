命令迁移

npm init -> yarn init
npm install -> yarn
npm install --save <name> -> yarn add <name>
npm install --save-dev <name> -> yarn add <name> --dev
npm update <name> -> yarn upgrade <name>
npm uninstall <name> -> yarn remove <name>
