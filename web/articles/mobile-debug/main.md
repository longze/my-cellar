# 移动端调试

> 移动开发需要的基础技能。

## 模拟器调试

使用 PC 浏览器的手机模式可以运行调试移动页面，方案简单调试流畅，缺点是无法定位真机上的 bug。

## 真机调试

### Safari(Mac) + iOS

手机打开“设置/Safari/高级/Web检查器”，Mac 打开 Safari/偏好设置(Preferences...)/高级选项(Advanced)。
