this、prototype、scope

学习如何在 js 里正确使用 this 就好比一场成年礼；原型是 js 实现继承等关系的重要基础；scope 则是闭包基础。

this

prototype

原型上的方法在 for of 中不会被遍历出来:

function A() {
    this.aa = 11;
}
A.prototype = {
    a: 1
};

const a = new A();

for (let key of Object.keys(a)) {
    console.log(key);
}
//输出: aa
